# AScript(working title)
A Unity game that uses AI to generate scripts that are acted out in the game. You can choose the characters, the setting, and the AI that's generating your script. Currently, the characters can only walk and talk, but soon they'll do a lot more! 

## Please note
This is currently in development, as there are only two characters and I haven't connected the AI Script and the actual scene so that I can make sure I can make the scene work with a test script. Please look out in the future for new releases for this game.
